How it works: edit the shop, gallery, numbers or phone below — you can upload photos or paste image links. When you are happy, press Save & publish to site in the bottom bar. Your changes are saved and go live on the website immediately for everyone. (Download backup saves a copy you can keep, and the site keeps working even if the server is briefly asleep.)

Shop products

These appear on the Shop page and the home page preview (first 4). Use the arrows to reorder; unticking "Show on site" hides a product without deleting it.

Impact numbers

Shown in the circles animation on the home page. Keep them short, like "22K+" or "13".

Contact phone

Replaces the "+91 XXXXX XXXXX" placeholder in the footer, pick-up page and contact page. Leave empty to keep the placeholder.

Gallery & Events

Photos and events shown on the Gallery page. Paste an image link (or leave it empty for a "photo coming soon" tile). Use the arrows to reorder.

Admin password

Change the password you use to open this dashboard. It must be at least 8 characters and include an uppercase letter, a lowercase letter, a number and a special symbol.

  • 8+ characters
  • Uppercase letter
  • Lowercase letter
  • Number
  • Special symbol
